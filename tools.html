Spotlight

MOSNA GUI

MOSNA GUI aims to make the MOSNA package more ergonomic for the detection of niches within biological tissue networks via a graphical interface. It allows modification of input parameters for both MOSNA and Tysserand to compute tissue networks from spatial transcriptomic/proteomic data and extract intrinsic statistical properties.

The interface handles the full workflow: from raw spatial coordinates to cell network construction, niche detection, assortativity computation, and cross-patient structural comparison.
